# fpgabuilder

Container based on [sbtscala/scala-sbt](https://hub.docker.com/r/sbtscala/scala-sbt) and [verilator/verilator](https://hub.docker.com/r/verilator/verilator) that bundles sbt, Verilator, and SpinalHDL.
